Detailed Description

template<typename Key>
class SkipList< Key >

Probabilistic skip list with unique keys.

Implements an ordered associative container with expected O(log n) complexity for insert, erase and search operations. Keys are stored in ascending order. Duplicate keys are ignored.

Template Parameters
Keytype of key, must be LessThanComparable (operator<)
